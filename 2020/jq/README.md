# 2020 Advent of Code
## JQ
```
# Usage aoc_jq_debug <NUMBER>
aoc_jq_debug() {
    watch -d -c "time jq --raw-input --color-output --slurp --from-file 2020/jq/${1}.jq 2020/input.${1} 2>&1"
}

# Usage aoc_jq_run <NUMBER>
aoc_jq_run() {
    jq --raw-input --slurp --from-file 2020/jq/${1}.jq 2020/input.${1}
}
```
